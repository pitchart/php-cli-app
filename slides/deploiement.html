Déploiement

Objectifs

  • Pouvoir télécharger l'application
  • Depuis un environnement sécurisé
  • Vérifier les données téléchargées

La solution Github

Github fournit une solution de gestion de pages pour les projets hébergés, basée sur une branche spécifique du dépot.

Github utilise HTTPS

Cloner le dépot

$ git clone https://github.com/[username]/[projectname].git

Créer une branche non rattachée à l'historique


$ cd [projectname]
$ git checkout --orphan gh-pages
$ git rm -rf .
$ rm .gitignore
https://[username].github.io/[projectname]

Vérifier les données téléchargées

Création d'un hash SHA1

$ sha1sum build/cli-app-demo.phar > build/cli-app-demo.phar.version

Vérification des données

$ sha1sum -c build/cli-app-demo.phar.version
